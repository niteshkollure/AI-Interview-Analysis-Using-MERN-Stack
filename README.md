# AI Interview Analysis

A simple React app for analyzing interview transcripts using basic AI, with login/signup functionality.

## Features

- Login/Signup page
- Input interview transcripts
- Sentiment analysis (positive, negative, neutral)
- Keyword extraction
- Left sidebar with image placeholder

## Prerequisites

- Node.js

## Setup

1. Clone or download the project.

2. Install backend dependencies:
   ```
   cd backend
   npm install
   ```

3. Install frontend dependencies:
   ```
   cd frontend
   npm install
   ```

4. Start the backend:
   ```
   cd backend
   npm start
   ```

5. Start the frontend:
   ```
   cd frontend
   npm start
   ```

6. Open http://localhost:3000 in your browser.

## Usage

Sign up or login, then enter an interview transcript and analyze. Results are shown temporarily (not saved).
